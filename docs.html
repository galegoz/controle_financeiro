Sobre o Aplicativo

O Controle Pessoal é um aplicativo robusto de gestão financeira desenvolvido em Flutter. O foco principal do projeto é garantir organização, desempenho e facilidade na manutenção e na escalabilidade do código. Para alcançar isso, todo o projeto foi estruturado seguindo os princípios de Clean Architecture, com uso do Provider para injeção de dependências e gerência de estado, e persistência local através do SQLite.

Arquitetura do Projeto (Clean Architecture)

A separação de responsabilidades é essencial para o desenvolvimento sustentável a longo prazo. O código fonte está dividido nas seguintes camadas principais dentro do diretório lib/:

⚙️
Core

Contém lógicas transversais, constantes, temas, formatações de moedas e datas, e paleta de cores (Design System).

📦
Domain

O coração da regra de negócio. Possui as Entidades (TransactionEntity, SettingsEntity) e os contratos/interfaces de Repositórios.

💾
Data

Responsável pela persistência. Contém os Modelos de Banco (conversão de JSON/Map), as implementações dos Repositórios e o DatabaseHelper do SQLite.

🎨
Presentation

Tudo relacionado à interface visual (UI) e estado. Contém Widgets genéricos, Páginas e os Providers (TransactionProvider, SettingsProvider, etc).

Fluxo de Dados

O fluxo de dados no aplicativo segue o modelo de via única:

  • A UI (Presentation) solicita uma ação chamando métodos nos Providers.
  • O Provider invoca as funções do Repositório (Domain).
  • A implementação do Repositório (Data) faz chamadas ao SQLite (DatabaseHelper), que mapeia a resposta para Entidades, e a devolve ao Provider.
  • O Provider notifica seus ouvintes (listeners), resultando na reconstrução reativa da Interface do Usuário.

Principais Funcionalidades

O aplicativo oferece um controle financeiro do zero ao nível avançado. Aqui estão as funcionalidades entregues:

  • 📊

    Dashboard Inteligente

    Visão geral instantânea que calcula as receitas e despesas correntes do mês, exibe o Saldo em tempo real, mostra um gráfico de pizza para comparação das entradas/saídas e fornece as movimentações mais recentes.

  • 🚨

    Controle de Orçamento (Progress Bar Visual)

    Acompanhe o limite mensal que você se propôs a gastar. A barra interage visualmente mudando de verde para amarelo e depois para vermelho conforme o limite definido for alcançado ou excedido.

  • 💳

    Cadastro Completo de Movimentações

    Formulários modernos e validados para o registro de Receitas ou Despesas. Inclui Data personalizada, Campo de Valor monetário, Tipo, Descrição e seleção da Forma de Pagamento.

  • 🔍

    Histórico com Pesquisa Dinâmica

    Filtre o histórico por "Despesas", "Receitas", ou pesquise pelo nome da movimentação para consultar os lançamentos antigos de forma instantânea.

  • 📈

    Relatórios, Gráficos e Insights

    Geração de relatórios por período personalizado. O aplicativo fornece um Gráfico de Barras com a evolução de entradas e saídas dos últimos 6 meses, além de apresentar "Insights Automáticos" baseados no comportamento do usuário (ex: "Você gastou X% a mais que no mês anterior").

  • 📤

    Exportação e Backup (PDF, Excel e CSV)

    As informações são 100% suas. Você pode exportar relatórios customizados no formato PDF, XLSX ou CSV para serem compartilhados (via e-mail, WhatsApp, etc). Na aba configurações, é possível realizar o Backup em PDF de toda a sua base de dados.

  • ⚙️

    Personalização Global

    Através do gerenciamento central de ThemeProvider e SharedPreferences, o usuário pode escolher entre o Tema Claro, Escuro ou do Sistema; definir a moeda de sua preferência (BRL, USD, EUR) e editar o orçamento base.

Bibliotecas e Ferramentas Utilizadas

  • Flutter SDK: Framework base multiplataforma.
  • Provider: Gerenciamento de Estado flexível, reativo e otimizado.
  • sqflite: Gerenciamento e persistência no banco de dados SQLite local.
  • fl_chart: Desenho e plotagem de Gráficos de Pizza e Barras (Altamente customizável).
  • shared_preferences: Armazenamento de dados leves, como temas e configurações simples.
  • pdf / excel / csv: Manipulação de arquivos para exportação dos dados contábeis.
  • share_plus: Integração com sistema operacional para compartilhamento dos relatórios gerados.
  • google_fonts: Integração dinâmica de fontes otimizadas.
